# LClient

### Descripción

Este proyecto lista todos los clientes registrados mostrando su fecha probable de muerte.

### Tecnologías Usadas
* Java 8
* Spring Boot
* Swagger
* Mongodb
* Lombok
* Log4J2

### Consumir Swagger
* dev -> http://localhost:9000/swagger-ui.html
* prod -> http://localhost:9000/lclient/swagger-ui.html
